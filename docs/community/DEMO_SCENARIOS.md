# Demo Scenarios

Use `cohrun` to launch local demos. Example scenario configs live in the
[`examples/`](../../examples/) directory.

## Kiosk Demo
```bash
cohrun kiosk_start
```

## Physics Demo
```bash
cohrun physics_demo
```

## Swarm Control
```bash
cohcli sim run examples/scenario_swarm_control.json
```
